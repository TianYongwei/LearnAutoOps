# 学习使用 Homebrew

## 是什么

> Package Manager for MacOS.

### 设计思想

- 软连接
- 已编译好的二进制包

### 基本概念

- 家酿
- 地窖
- 酒桶
- 酒瓶
- 配方

## 怎么用（基于场景）

- 搜索关键字相关的可用App
- 查看已经安装的App列表
- 安装App
- 卸载App
- 显示App依赖
